Project: Forum API

2022-01-07 :: Larry Flint
#projects  #django  #postgres 
This is a database for a user-generated chat forum that is ready to be paired with a front-end UI. It uses the Django-Ninja framework, Postgres, and cookie based authentication. Intended for use as a technology forum, but is suitable for multiple industries. This project started off as a full stack app, intending to be my capstone project for my bootcamp. It was meant to be a technology forum for me and my cohorts to compile our resources.
